1. 修改 `.vscode\launch.json` 
  ```
    "configurations": [
          {
              "name": "Flutter",
              "type": "dart",
              "request": "launch",
              "program": "lib/main.dart",
              "cwd": "${workspaceFolder}",
              "args": ["--dart-define=API_BASE_URL=http://127.0.0.1:8000"] ← 加這行
          },
      ]
  ```
2. 終端機輸入 `uvicorn app.main:app --reload`
